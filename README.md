# TaskBoard [[Download]](https://github.com/EhsanFadhli/TaskBoard/releases)
Simple app that can transforms plain .txt markdown files into structured, sortable task lists. With simple intuitive UI, it parses markdown syntax and converts them into organized tasks I can easily track and manage.
